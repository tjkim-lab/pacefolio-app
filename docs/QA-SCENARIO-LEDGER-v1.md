> 판정값: PASS-AUTO / PASS-MANUAL / FAIL / BLOCKED / NOT-IMPLEMENTED / NOT-TESTED.

> ASYNC/NOTIFICATION/AUDIT/RECONCILE/RECOVERY) 2축 — "백엔드 계층 PASS ≠ 생애주기 PASS".
> LCV1-P0-01·02(웹 경계·noindex)=proxy.ts+metadata 로 mock 단계 조치 완료(실 세션 검사는
