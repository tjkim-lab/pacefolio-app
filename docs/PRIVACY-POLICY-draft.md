# PACEFOLIO 개인정보처리방침 (초안 v0.1 — 법률 검토 전, 게시 금지)

> ⚠️ 이 문서는 개발 단계 초안이다. 변호사 검토·TJ 승인 후 확정본을
> 랜딩 페이지와 앱 설정에 게시한다. [ ] 표시는 확정 필요 항목.

**시행일**: [확정 시 기입] · **개인정보보호책임자(CPO)**: 김태정 (tj.kim@wondergym.co.kr)

## 1. 처리 목적과 지위
PACEFOLIO(운영: [사업자명])는 유소년 스포츠·교육 아카데미 운영 플랫폼으로,
① **학원이 위탁한 원생·수업 정보의 수탁 처리자**이자
② **보호자·직원 계정 정보의 개인정보처리자**입니다. (혼합 지위 — docs/16 §A)

## 2. 수집 항목과 방법
| 구분 | 항목 | 방법 |
|---|---|---|
| 보호자 계정 | 이름·휴대전화(인증)·소셜 계정 식별자(카카오 등)·이메일(선택) | 소셜 로그인·회원가입 |
| 원생(만 14세 미만) | 이름·생년월일·소속 반·출결·성장 활동 기록 | **학원(처리자) 등록 + 법정대리인(보호자) OTP 연결 동의** |
| 건강정보(민감) | 알레르기·부상·컨디션 메모 | **별도 동의** 후 학원·담당 코치 입력 |
| 사진 | 수업 사진(원생 식별 가능) | **목적·대상별 동의**(전 원생 동의 없는 사진은 공유 제외) |
| 결제 | 청구·결제·환불 내역 (⚠️ **카드번호는 수집·저장하지 않음** — 결제대행사 처리) | 결제 시 |
| 자동 수집 | 접근 로그·기기 정보 | 서비스 이용 시 |

## 3. 만 14세 미만 아동
원생 정보는 법정대리인(보호자)의 동의를 얻어 처리합니다. 동의는 학원 선등록
연락처 또는 초대코드와 결합된 **휴대전화 본인인증(OTP) 기반 보호자-자녀 연결
절차**로 획득·기록되며, 보호자는 언제든 연결을 철회할 수 있습니다(철회 이력 보존).

## 4. 보유·이용 기간과 파기
docs/16 §B 표를 따릅니다. 요지: 결제·계약 기록 5년(전자상거래법) ·
원생 기록 퇴원 후 1년 · 건강 메모 퇴원 시 파기 · 사진 게시 후 1년 자동 삭제 ·
탈퇴 시 즉시 파기(법정 보존 항목만 분리보관). 기간 경과 시 복구 불가능한
방법으로 파기합니다.

## 5. 제3자 제공·처리위탁
- 제3자 제공: 하지 않습니다(법령 근거 또는 별도 동의 시 예외).
- 처리위탁: [결제대행사 — 확정 시 기입] (결제 처리), [클라우드 — 확정 시 기입] (인프라),
  [알림 발송사 — 확정 시 기입]. 위탁 계약으로 안전조치를 감독합니다.

## 6. 정보주체의 권리
보호자는 열람·정정·삭제·처리정지·동의 철회를 요구할 수 있습니다(앱 설정 또는
CPO 이메일). 만 14세 미만 원생의 권리는 법정대리인이 행사합니다.

## 7. 안전성 확보 조치
역할·학원 단위 접근권한 분리 / 접근·처리 감사 기록 / 세션 토큰 해시 저장 /
전송 구간 암호화(TLS) / 주요 개인정보 저장 암호화(전화번호 등 — 단계 적용) /
금액·건강정보의 잠금화면·알림 비노출 / 결제카드정보 미보유.
