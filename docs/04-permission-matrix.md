# 04 · 권한 매트릭스 · 멀티테넌트 격리 (리뷰 P0-6)

누가 무엇을 할 수 있고, 어떤 데이터를 볼 수 있나. 코드: `packages/domain/permissions.ts`.
> **2단계 검증(서버):** ① 역할이 능력을 가지는가 `can()` **AND** ② 대상이 같은 학원·본인 소유인가(scope). 역할 통과는 **필요조건일 뿐.**
> **클라이언트가 준 `role`·`academyId`·`ownerId`·`participantId`·`permission`·`supportViewEnabled` 은 절대 신뢰하지 않는다.** 서버가 세션에서 도출.

**핵심 규칙**
- **코치는 금액을 못 본다**(금액=개인정보, 헌법). 코치 앱엔 수강료·정산 없음.
- **코치는 정책을 못 바꾼다** — 읽기전용. (리뷰 6-3: "편집권한 시연 토글" 실제품 제거.)
- **환불은 학부모 요청 + 원장 승인**(상호). 어느 한쪽만으로 실행 불가(헌법).
- **관리자(플랫폼)는 테넌트 데이터를 직접 수정하지 않는다** — 지원 뷰(읽기·마스킹·15분·감사)로만.

## 2. 필드 공개 범위 (SNS식 Everyone 금지, 리뷰 4-2)
민감 필드는 사용자가 임의로 "전체공개" 불가 — 시스템 정책(`PrivacyVisibility`)으로 고정.

| 필드 | 공개 범위 |
|---|---|
| 코치 이름·프로필사진 | 소속 학원 보호자 |
| 코치 휴대전화 원문 | 원장·승인 관리자만 (그 외엔 중계전화·인앱채팅) |
| 보호자 전화번호 | 학원 관리자, 필요 시 담당 코치 |
| 원생 생년월일 | 보호자·승인된 학원 담당자 |
| **원생 건강·안전정보** | 담당/대체 코치·원장·승인 안전담당 **최소 인원** (+조회기록) |
| 주소 | 기본 수집 안 함 |
| **결제 금액** | 보호자(본인)·원장·승인 수납담당 |
| 사진 동의 | 보호자·담당 학원 관리자 |

## 3. 멀티테넌트 격리 (리뷰 P0-6)
모든 서버 요청에서 순서대로 검증:
```
로그인 사용자 → 역할(서버 도출) → 소속 조직 → 접근가능 academyId 집합
   → 대상 리소스의 academyId 일치?  (inTenantScope)
   → 해당 능력 보유?               (can)
   → (보호자/코치) 연결·담당 스코프? (link/assignment 확인)
```
- `academyId` 는 거의 모든 엔티티에 존재 = 격리 축. 대상 리소스의 academyId ∉ 행위자 academyId 집합 → **403**.
- 보호자 능력은 **검증(VERIFIED)된 GuardianParticipantLink** 로 연결된 자녀에만. 코치의 건강정보·출결확정은 **담당(ClassAssignment)** 반으로만.

## 4. 반드시 통과할 부정 테스트 (리뷰 P0-6)
- [ ] 다른 학원의 `participantId` 직접 요청 → 403
- [ ] 다른 보호자의 자녀 조회 → 403
- [ ] 퇴사(ENDED) 코치의 기존 토큰 재사용 → 401 (세션 만료)
- [ ] **코치가 결제 금액 조회 → 403**
- [ ] 보호자가 다른 가족 청구서 조회 → 403
- [ ] 원장이 Admin(플랫폼) API 호출 → 403
- [ ] Support View 만료(15분) 후 재요청 → 401
- [ ] 마스킹된 필드를 URL 파라미터 변경으로 우회 → 여전히 마스킹/403
- [ ] 클라이언트가 `role: OWNER` 위조 전송 → 무시(서버 세션 기준)

## 5. 코드 사용 예
```ts
import { can, inTenantScope } from "@pacefolio/domain";

// 코치가 금액 보기? → false
can("COACH", "VIEW_PAYMENT_AMOUNT"); // false
can("OWNER", "APPROVE_REFUND");      // true

// 대상 학원이 내 소속인가
if (!inTenantScope(session.academyIds, target.academyId)) throw forbidden();
// + 보호자/코치면 링크·담당 스코프 추가 확인
```
